## Installation

Install dependencies:

````
pnpm run install
````

Start compiling and running a [local server](https://developer.mozilla.org/en-US/docs/Learn/Common_questions/Tools_and_setup/set_up_a_local_testing_server):

````
pnpm run dev
````

Building application:

````
pnpm run build
````
